Ce projet est une application web développée avec le framework Django  permettant la gestion d’une bibliothèque simple.
Elle permet aux utilisateurs authentifiés de gérer une liste de livres (ajout, modification, suppression et recherche).

Le système inclut également un module d’authentification (inscription, connexion et déconnexion).
